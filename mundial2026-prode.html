PASO 1 DE 4

🔥 Crear proyecto en Firebase (gratis)

Firebase es la base de datos que va a guardar usuarios, predicciones y ranking en tiempo real. Es 100% gratuito para proyectos como este.

→ Andá a console.firebase.google.com
→ Hacé clic en "Crear un proyecto"
→ Nombre: prode2026 → Desactivá Google Analytics → Crear proyecto

PASO 2 DE 4

🔑 Obtener las credenciales

Dentro de tu proyecto Firebase → hacé clic en el ícono </> (Web) → registrá la app como "prode2026" → copiá el bloque firebaseConfig que aparece.

Vas a necesitar estos valores: apiKey, authDomain, projectId, storageBucket, messagingSenderId, appId

PASO 3 DE 4

👤 Activar autenticación

En Firebase Console → Authentication → "Comenzar" → Método de acceso → activá "Correo electrónico/Contraseña" → Guardar.

PASO 4 DE 4

🗄️ Crear la base de datos (Firestore)

En Firebase Console → Firestore Database → "Crear base de datos" → elegí "Comenzar en modo de prueba" → Ubicación: us-central → Listo.
